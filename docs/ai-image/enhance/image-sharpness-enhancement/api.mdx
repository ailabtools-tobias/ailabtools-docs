## Request

- **URL**: `https://www.ailabapi.com/api/image/enhance/image-sharpness-enhancement`
- **Method**: `POST`
- **Content-Type**: `multipart/form-data`

### Image requirements

- **Image format**: `PNG` `JPG` `JPEG` `BMP`
- **Image size**: No more than 8 MB.
- **Image resolution**: Larger than 10x10px, smaller than 5000x5000px.
- **Image aspect ratio**: Aspect ratio within 4:1.

### Headers

| Field              | Required | Type     | Description                                        |
|:-------------------|:---------|:---------|:---------------------------------------------------|
| `ailabapi-api-key` | YES      | `string` | Application API KEY. [Get API KEY](/doc/get-api-key) |

### Body

| Field   | Required | Type   |
|:--------|:---------|:-------|
| `image` | YES      | `file` |
