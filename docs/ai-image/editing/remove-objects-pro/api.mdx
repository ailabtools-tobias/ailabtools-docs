## Request

- **URL**: `https://www.ailabapi.com/api/image/editing/remove-objects-pro`
- **Method**: `POST`
- **Content-Type**: `multipart/form-data`

### Image requirements

- **Image format**: `JPEG` `JPG` `PNG` `BMP` `WEBP`
- **Image size**: No more than 5 MB.
- **Image resolution**: Less than 4096x4096px.

### Headers

| Field              | Required | Type     | Description                                        |
|:-------------------|:---------|:---------|:---------------------------------------------------|
| `ailabapi-api-key` | YES      | `string` | Application API KEY. [Get API KEY](/doc/get-api-key) |

### Body

| Field   | Required | Type   | Description     |
|:--------|:---------|:-------|:----------------|
| `image` | YES      | `file` | Original image. |
| `mask`  | YES      | `file` | Mask image.     |
