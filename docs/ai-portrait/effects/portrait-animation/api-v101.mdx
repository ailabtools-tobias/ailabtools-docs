## Request

- **URL**: `https://www.ailabapi.com/api/portrait/effects/portrait-animation`
- **Method**: `POST`
- **Content-Type**: `multipart/form-data`

### Image requirements

- **Image format**: `JPEG` `JPG` `PNG`
- **Image size**: No more than 3 MB.
- **Image resolution**: Larger than 100x100px, smaller than 2000x2000px.

### Headers

| Field              | Required | Type     | Description                                        |
|:-------------------|:---------|:---------|:---------------------------------------------------|
| `ailabapi-api-key` | YES      | `string` | Application API KEY. [Get API KEY](/doc/get-api-key) |

### Body

| Field   | Required | Type     | Scope                 |
|:--------|:---------|:---------|:----------------------|
| `image` | YES      | `file`   |                       |
| `type`  | YES      | `string` | [More Details](#type) |
