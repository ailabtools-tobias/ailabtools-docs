| Field    | Type     | Description        |
|:---------|:---------|:-------------------|
| `data`   | `object` | Final result data. |
| +`image` | `string` | Result image URL.  |

```json
{
  "data": {
    "image": ""
  }
}
```

<Tip>
  `image` is temporary and remains valid for 24 hours. Download the file to your own storage before the URL expires if you need long-term storage.
</Tip>
