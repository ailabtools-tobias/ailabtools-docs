## Request

- **URL**: `https://www.ailabapi.com/api/portrait/effects/face-tidyup`
- **Method**: `POST`
- **Content-Type**: `multipart/form-data`

### Image requirements

- **Image format**: `JPEG` `JPG` `PNG` `BMP`
- **Image size**: No more than 3 MB.
- **Image resolution**: Larger than 10x10px, smaller than 2000x2000px.
- **Image quality recommendation**: the input image should be well-lit and evenly lit as much as possible, the face should be intact and less obscured, and the angle [raw, yaw, pitch] should be less than 60 degrees.
