| Field       | Type     | Description                |
|:------------|:---------|:---------------------------|
| `data`       | `object` | Final result data.        |
| +`image_url` | `string` | Result image URL address. |

```json
{
  "data": {
    "image_url": ""
  }
}
```

<Tip>
  `image_url` is temporary and remains valid for 24 hours. If you need long-term storage, download the file to your own storage within that period.
</Tip>
