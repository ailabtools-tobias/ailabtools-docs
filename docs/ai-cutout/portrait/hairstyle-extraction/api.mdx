## Request

- **URL**: `https://www.ailabapi.com/api/cutout/portrait/hairstyle-extraction`
- **Method**: `POST`
- **Content-Type**: `multipart/form-data`

### Image requirements

- **Image format**: `JPEG` `JPG` `BMP` `PNG`
- **Image size**: No more than 3 MB.
- **Image resolution**: Larger than 32x32px, smaller than 2000x2000px.

### Headers

| Field              | Required | Type     | Description                                        |
|:-------------------|:---------|:---------|:---------------------------------------------------|
| `ailabapi-api-key` | YES      | `string` | Application API KEY. [Get API KEY](/doc/get-api-key) |

### Body

| Field   | Required | Type   |
|:--------|:---------|:-------|
| `image` | YES      | `file` |
