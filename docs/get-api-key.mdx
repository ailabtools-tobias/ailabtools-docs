---
title: "Get API KEY"
---

Follow the steps below to create and retrieve your API Key.

---

## Step 1 — Create an Account

<a href="https://www.ailabtools.com/account/signup" target="_blank">
  Create Account
</a>

Create an AILabTools account to access the developer platform and API services.

If you already have an account, you can skip this step.

![Create Account Image](/images/get-api-key/1.webp)

---

## Step 2 — Sign In to Developer Platform

<a href="https://www.ailabtools.com/developer" target="_blank">
  Open Developer Platform
</a>

Sign in to the developer platform to access API services and developer features.

> APIs do not require separate activation or approval and can be used immediately after obtaining an API Key.

![Developer Platform Image](/images/get-api-key/2.webp)

---

## Step 3 — Complete Email Verification

Complete email verification before creating API Keys.

> You must complete email verification before receiving free API credits or creating API Keys.

![Complete Email Verification Image 1](/images/get-api-key/3-1.webp)
![Complete Email Verification Image 2](/images/get-api-key/3-2.webp)

---

## Step 4 — Receive Free API Credits

After successfully completing email verification, free API credits will be automatically granted to your account for testing supported APIs.

![Free API Credits Image](/images/get-api-key/4.webp)

---

## Step 5 — Create API Key

<a href="https://www.ailabtools.com/developer/keys" target="_blank">
  Open API Keys Page
</a>

Create a new API Key for your application or project.

You can create different API Keys for different environments, such as:

- Production
- Development
- Testing

![Open API Keys Page Image](/images/get-api-key/5-1.webp)

![Create API KEY Image](/images/get-api-key/5-2.webp)

![API KEY Configuration Image](/images/get-api-key/5.3.webp)

---

## Step 6 — Get API Key

After the API Key is created, copy and securely store your API Key.

![Get API KEY Image](/images/get-api-key/6.webp)
